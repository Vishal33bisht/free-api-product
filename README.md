# 🛍️ Free API Product App

A modern product listing web application built using HTML, CSS, and JavaScript.  
This project demonstrates API integration, dynamic product rendering, and responsive UI development using vanilla JavaScript.

---

## 🚀 Features

- 📦 Fetch products dynamically from API
- 🖼️ Display product images, prices, and details
- 📱 Fully responsive design
- ⚡ Fast and lightweight frontend
- 🔍 Dynamic product rendering using JavaScript
- 🎨 Clean and modern UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- REST API

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Vishal33bisht/free-api-product.git
```

Move into the project directory:

```bash
cd free-api-product
```

Open the project in your browser:

```bash
open index.html
```

Or simply double-click `index.html`.
